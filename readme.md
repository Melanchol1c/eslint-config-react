# ESLint configuration for react projects

## Installation
```
npm i -D eslint-config-react-kirizey
```

## Usage
### .eslintrc

```
{
    "extends": "eslint-config-react-kirizey"
}
```

### .prettierrc
```
{
  "printWidth": 120,
  "tabWidth": 2,
  "semi": true,
  "trailingComma": "all",
  "bracketSpacing": true,
  "jsxBracketSameLine": false,
  "singleQuote": true
}
```

### .browserslistrc
```
> 1%
last 2 versions
```
